Proyecto Final: Juego Nave Espacial
===================================
Proyecto Simple un juego de una nave que dispara a asteroides.


Empezando
=========
Para correr el programa solo se necesita un compilador de C++.


Pre-requisitos
==============
El programa corre en cualquier sistema de 32gb o 64gb con un programa que 
compile código de C++.

Instalación
=========== 
Descargar el código de: https://github.com/joshe1129/Ciencia_Computacion_1/tree/master/Nave_ProyectoFinal
Y correrlo en el compilador de C++ de su gusto.

Ejecución
=========
El juego funciona con las teclas direccionales y la tecla ‘A’ para disparar.

Construido con
==============
Código en lenguaje de programación C++ con Code::Blocks
